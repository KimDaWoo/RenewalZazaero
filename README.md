# Zazearo

## 실행하기 전

클론을하여 파일을 다운받은 경우에는 하단의 방법으로 패키지를 설치한 후 실행합니다.

```bash
# step 1
npm install

# step 2
npx pod-install

# step 3
npm start
```

## 규칙

### 작성규칙

1. 폴더명은 대문자로 시작한다.
2. 함수는 카멜케이스로 작성한다.
3. 클래스 명은 대문자로 시작한다.
